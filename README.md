# 🚀 Auto Job Applyer (Python 自动投简历助手)
- 一个基于 Python 的自动化脚本，帮助你在主流招聘网站上批量、自动地投递简历，解放双手，提高求职效率

# ⚠️ 免责声明
- 本项目仅供学习交流和技术研究使用。
- 请勿利用本脚本对招聘平台进行恶意攻击或高频骚扰。
- 频繁的自动化操作可能会导致你的账号被平台风控系统限制或封禁。
- 使用本脚本所产生的一切后果（包括但不限于账号封禁、法律纠纷等），由使用者本人承担，与开发者无关。

# ✨ 功能特性
- 多平台支持：目前支持 Boss直聘，其他平台正在适配
- 技术实现：主要是利用模拟计算机操作，暂时不涉及爬虫
